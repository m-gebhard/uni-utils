    DrawLine(Nullable<Color>, Int32, Int32)

    Draws a horizontal line in the Unity Editor with customizable color, thickness, and padding.

    Declaration
    public static void DrawLine(Color? color = null, int thickness = 1, int padding = 10)
    Parameters
    Type Name Description
    System.Nullable<Color> color

    The color of the line. Defaults to gray if not specified.
    System.Int32 thickness

    The thickness of the line in pixels. Defaults to 1.
    System.Int32 padding

    The vertical padding around the line in pixels. Defaults to 10.
    Examples
    // Draw a red line with thickness 2 and padding 5
EditorGUILayoutExtension.DrawLine(Color.red, 2, 5);
